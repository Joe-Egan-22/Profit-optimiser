# Profit optimser

Maxmimises profit using scipy linprog. Does so based on constraints in materials and time. Useful for small one-off events (e.g. bake sales, charity events, early business start-up). Impractical for larger systems (e.g. production companies).

Code can be downloaded and used straight away, so long as python is available on the device.
User inputs the number of products in consideration, manufacturing monetary cost, sell price, material cost, amount of available materials, manufacturing time and maximum time available.
Code will output the maximum profit which can be obtained and the number of each product to be made.

Not currently updating this project, possibly will adapt for larger-scale systems by increasing the constraint considerations.
